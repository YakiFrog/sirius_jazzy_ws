# 自律移動中断
```bash
ros2 lifecycle set /bt_navigator deactivate
```

# 自律移動停止
```bash
ros2 service call /navigate_to_pose nav2_msgs/srv/CancelGoal "{}"
```

# rviz satelliteの導入
```bash
sudo apt install ros-jazzy-rviz-satellite
```
デモの起動￥
```bash
ros2 launch rviz_satellite demo.launch.xml
```
Rviz2の起動
```bash
rviz2 -d demo.rviz
```
