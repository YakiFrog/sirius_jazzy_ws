# デバッグツール
```bash
sudo apt update
sudo apt install ros-jazzy-ros2trace ros-jazzy-tracetools-analysis
```
ros-jazzy-ros2trace はROS 2アプリケーションの実行時にトレースデータを取るためのツールを提供します。主にプログラムの動作遅延やコールバック処理時間などの性能解析に使います。

ros-jazzy-tracetools-analysis は収集したトレースデータを解析・可視化するためのツール群です。Pythonベースの解析APIやJupyter Notebookでの解析例もあります。
